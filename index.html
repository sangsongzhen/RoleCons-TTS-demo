语音合成 Demo

下表展示了 Ground-Truth、DailyTalk、Proposed Model 三种系统在不同对话下的生成效果。

对话文本 Ground-Truth DailyTalk Proposed Model
Dialog 1: I am looking for a pan.
Dialog 2: No problem. What size would you like?
Dialog 3: A big one would be nice.
Dialog 4: How about this one? It's our biggest — sixteen' in diameter.
Dialog 5: Oh, yes, I like that one, but it's too heavy.
Dialog 6: OK, try this one. It's made of aluminum.
Dialog 7: Oh, yes! This is much better. But it has an aluminum handle.
Dialog 8: Here you go. Same pan, but with a state-of-the-art, heat-resistant plastic handle.
Dialog 9: Yes. That's perfect. I'll take it.
Dialog 10: Great. Will that be cash or credit?
Dialog 11: Oh, wait a minute. What about a lid for the pan?
Dialog 12: I'm sorry. I forgot to show you the lid. It comes with the pan.
